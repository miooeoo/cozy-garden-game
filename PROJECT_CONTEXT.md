# 🌷 코지 가든 - 프로젝트 컨텍스트 (V3.0)

## 개요
치유형 픽셀 아트 가드닝 시뮬레이션 게임. "실패 없음, 풍요로움, 안전함"을 핵심 철학으로 합니다.

**배포 링크**: https://miooeoo.github.io/cozy-garden-game/

---

## 파일 구조

### 핵심 시스템
| 파일 | 역할 |
|------|------|
| `game.js` | 메인 게임 클래스, 루프, UI 이벤트 |
| `Garden.js` | 정원 그리드, 식물 관리, 저장/로드 |
| `Plant.js` | 식물 클래스, 성장 단계, PlantTypes 정의 |
| `Character.js` | 농부 캐릭터, 클릭투무브, 애니메이션 |
| `Inventory.js` | 골드/씨앗/작물 관리 |

### 게임플레이 시스템
| 파일 | 역할 |
|------|------|
| `Shop.js` | 상점 UI, 구매/판매 |
| `ShippingBin.js` | 배송 상자, 일일 정산 |
| `MarketSystem.js` | 동적 시세, 인기 아이템 |
| `DragDropSystem.js` | 드래그 수확 |
| `ObstacleManager.js` | 랜덤 바위 스폰/제거 ⭐NEW |

### 특수 시스템
| 파일 | 역할 |
|------|------|
| `RainCloudSystem.js` | 광고 보상 비구름 |
| `JournalSystem.js` | 식물 도감 |
| `MutationManager.js` | 돌연변이/희귀 식물 |

### UI/스타일
| 파일 | 역할 |
|------|------|
| `index.html` | 게임 레이아웃, 스크립트 로드 |
| `styles.css` | 체리블라썸 드림 테마 |

---

## 주요 조작법

| 입력 | 동작 |
|------|------|
| 클릭 (빈 타일) | 캐릭터 이동 → 도착 후 심기 |
| WASD/화살표 | 직접 이동 |
| 드래그 | 다 자란 작물 연속 수확 |
| P키 | 자동 파종 모드 토글 |
| 📦 택배 탭 | 즉시 판매 UI |

---

## 최근 업데이트 (V3.0)

1. **농부 캐릭터**: 밀짚모자+멜빵바지, 4방향 걷기/심기 애니메이션
2. **클릭 투 무브**: 타일 클릭 → 이동 → 도착 후 심기
3. **퀵 택배 UI**: 우측 고정 탭, 즉시 판매 모드
4. **랜덤 바위**: 1분마다 스폰, 2분 후 자연 소멸 또는 클릭 제거

---

## 기술 스택

- **순수 JavaScript** (ES6 클래스)
- **Canvas 2D** (픽셀 아트 렌더링)
- **localStorage** (세이브 데이터)
- **GitHub Pages** (배포)
